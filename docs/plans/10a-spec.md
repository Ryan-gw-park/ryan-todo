## 1. 목표

1. **개인 매트릭스**: MS를 'today' 컬럼에서만 → **모든 컬럼에서 표시** (팀 매트릭스와 동일)
2. **매트릭스 전체 (개인+팀)**: MS 행을 시각적 앵커로 강화 — **배경색 + 접이식 그룹 헤더 + task 들여쓰기**

---

## 2. 확정 결정사항

| # | 항목 | 결정 |
|---|------|------|
| D1 | 개인 매트릭스 MS 표시 | 모든 컬럼에서 표시 (today 제한 제거) |
| D2 | InlineMsAdd (+ 마일스톤 추가) | **today 컬럼만 유지** |
| D3 | MS 행 배경 | 연한 배경색 (항상 표시, hover 아닐 때도) |
| D4 | MS 하위 task | 왼쪽 들여쓰기 추가 |
| D5 | 접기/펼치기 | 기존 기능 유지 + 접힌 상태에서 `▸ MS명 (N)` compact 표시 |
| D6 | DB 변경 | 없음 |
| D7 | 주간 플래너 영향 | 없음 (MilestoneRow는 매트릭스 셀 전용) |

## 3. 기능 범위

### 3-1. 개인 매트릭스 MS 전체 컬럼 표시

- PersonalMatrixGrid line 65: `cat.key === 'today'` 조건 제거
- 모든 카테고리 컬럼 (today, next, later)에 MS 그룹 표시
- InlineMsAdd는 today만 유지 (D2)

### 3-2. MilestoneRow 시각 강화

**현재**: 투명 배경, 11px 600 weight, 회색 bullet, task와 구분 약함

**변경**:
- **배경**: `#f5f4f0` (연한 회색, 항상 표시) — hover 시 약간 더 진하게
- **좌측 accent**: 내부 4px 세로 div (프로젝트 dot 색상) — border-left 금지 규칙 준수
- **폰트**: 12px bold → 기존 11px 600보다 약간 크게
- **카운트 badge**: `(N)` → pill 배경 추가
- **패딩**: 상하 5px (기존 3px 2px보다 넓게)
- **접힌 상태**: chevron 회전 + task 숨김 (기존 동작 유지)

### 3-3. Task 들여쓰기

- CellContent에서 MS 그룹 하위 task 렌더 시 `paddingLeft: 14px` 추가
- ungrouped task (MS 없는)는 들여쓰기 없음
